### Пример практического задания на экзамен к дисциплине "Информатика"
```
Студент купил в книжном магазине следующие книги:
   - "Война и мир" - 3 книги за 450 рублей
   - "Преступление и наказание" - 1 книга за 470 рублей
   - "Мастер и Маргарита" - 2 книги за 520 рублей

1) найдите картинку на тему "Книжный магазин" и сохраните в созданную на диске
(в каталоге "Документы") папку с таким же названием " Книжный"

2) в Visual Studio Code откройте ранее созданную папку и создайте html-страницу ("!+tab")
с заголовком страницы "Сумма покупок" и тем же заголовком 3 уровня тексте на странице,
с найденной картинкой и таблицей из 3 столбцов (Наименование, Кол-во, Цена) и 3 строк.
Сохраните документ в ранее созданную папку. Запустите в браузере через кнопку «Go Live».

3) создайте Word-документ с заголовком "Сумма покупок"
(полужирный шрифт, размер 18, выравнивание по центру),
с найденной картинкой и ненумерованным списком покупок.
Сохраните документ в ранее созданную папку.

4) создайте Excel-документ с заголовком "Сумма покупок"
из 5 столбцов (Наименование, Кол-во, Ед.изм, Цена, Сумма) и 3 строк.
Сохраните документ в ранее созданную папку.
Выполните:
- в столбце Сумма задайте формулу подсчета общей суммы купленного товара
- в отдельной итоговой ячейке подсчитайте общую сумму покупок
- в отдельном столбце подсчитайте процент каждого товара от общего количества
- постройте круговую диаграмму (с % на долях).
```
